# Trashfish

![Cover Art](https://github.com/Spectryl/Trashfish/blob/main/art/art%20project%20files/cover_art.png?raw=true)

Trashfish is a fast-paced arcade score game placing you as a simple fish who is starving due to humanity's pollution of the oceans.

Clean up the ocean while dodging humanity's trash (and other pollution) and restore the ocean to its proper status.
